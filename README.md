Pizza Sales Performance Dashboard | Power BI

- Project Overview
This project presents an interactive Pizza Sales Performance Dashboard built using Power BI. The dashboard analyzes over 21,000 sales transactions to evaluate revenue trends, order behavior, time-based distribution, and category-level performance.
The objective of this project is to transform raw transactional data into actionable business insights through KPI-driven reporting and data visualization.

- Business Objectives
1. Monitor overall revenue performance
2. Analyze order distribution by day and time
3. Identify top-performing product categories
4. Evaluate monthly revenue trends
5. Design executive-level KPIs for decision support

- Key Performance Indicators (KPIs)
1. Total Revenue: ₹817K
2. Total Orders: 21K
3. Average Order Value (AOV): ₹38.31
4. Total Quantity Sold: 50K

- Key Insights Derived
1. July recorded the highest monthly revenue.
2. Friday generates the maximum number of orders.
3. Afternoon accounts for the majority of total sales.
4. Classic category contributes the highest revenue share.

- Technical Implementation
1. Built using Power BI Desktop
2. Created DAX measures for:
3. Revenue aggregation
4. Average Order Value (AOV)
5. Ranking calculations
6. Time-intelligence analysis
7. Implemented dynamic slicers and conditional formatting
8. Designed business-focused KPI layout and visual storytelling

- Dashboard Features
1. Monthly revenue trend analysis
2. Orders by day of week
3. Time-based sales distribution
4. Revenue by category
5. Interactive filtering via slicers
6. Insight summary panel
